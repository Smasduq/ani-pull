Ani-Pull APT Repository

Welcome to the official APT repository for ani-pull.

How to add this repository:

  1. Install the GPG key:
    wget -qO - https://smasduq.github.io/ani-pull/repo/key.gpg | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/ani-pull.gpg
  2. Add the source:
    echo "deb https://smasduq.github.io/ani-pull/repo/ bookworm main" | sudo tee /etc/apt/sources.list.d/ani-pull.list
  3. Install:
    sudo apt update && sudo apt install ani-pull

Supported distributions: bookworm, bullseye, trixie, jammy, noble
